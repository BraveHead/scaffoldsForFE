# textEditor

本文本编辑器使用 css/js/jquery 实现界面及功能，并使用 webpack 进行打包。运行依赖 node.js 环境。

## 运行

进入 `webpack.config.js` 所在目录，在控制台运行 `npm start`，在浏览器输入：`localhost:9000`，即可显示文本编辑器界面。

ps：如当前目录已经存在 `node_modules` 文件夹，则忽略该说明。如当前目录下没有 `node_modules` 文件夹，则先运行 `npm install` 安装相关包文件，再运行上述步骤。
